# 🚀 Monte Carlo Insight Simulator

불확실한 변수들 사이에서 최선의 의사결정을 내릴 수 있도록 돕는 확률 기반 분석 도구입니다.

## 1. 프로젝트 개요
사용자가 입력한 변수의 범위와 분포를 바탕으로 수만 번의 시뮬레이션을 실행하고, 결과의 90% 신뢰 구간 및 민감도를 분석하여 시각화된 보고서를 제공합니다.

## 2. 주요 기능 및 요구사항 (Enhanced)

### F-01. 유연한 변수 설정
- **다중 변수 입력**: 변수명, 최솟값, 최댓값, 선호하는 분포(균등, 정규, 삼각분포)를 설정 가능.
- **시나리오 관리**: '낙관', '중립', '비관' 등 시나리오 세트를 저장하고 불러오는 기능.

### F-02. 고도화된 시뮬레이션 엔진
- **자동 수렴 감지**: 사용자가 횟수를 정하지 않아도 결과값이 안정화될 때까지 자동으로 시뮬레이션 수행 (옵션).
- **상위/하위 90% 수치 계산**: 결과 데이터의 5th 및 95th 백분위수를 계산하여 안전 범위를 제시.

### F-03. 인터랙티브 시각화 (Insights)
- **확률 분포도 (Histogram)**: 결과값의 빈도와 90% 신뢰 구간을 하이라이트 표시.
- **수렴 그래프 (Convergence Plot)**: 시뮬레이션 횟수에 따른 평균값의 변화를 보여주어 결과의 신뢰성 입증.
- **토네이도 차트 (Sensitivity Analysis)**: 어떤 입력 변수가 결과에 가장 큰 영향을 미쳤는지 중요도순으로 정렬.



## 3. 사용자 인터페이스 (UI) 계획
- **입력단**: 사이드바를 통해 변수 추가 및 시뮬레이션 횟수 설정.
- **결과단**: 메인 화면 상단에 '90% 확률로 예상되는 최저/최고치'를 요약 카드로 표시.
- **차트**: Plotly 또는 Seaborn을 활용하여 인터랙티브한 그래프 배치.

## 4. 기술 스택
- **Language**: Python 3.9+
- **Core**: NumPy (수치 계산), Pandas (데이터 정리)
- **Viz**: Streamlit (웹 UI), Plotly (인터랙티브 차트)

## 5. 핵심 알고리즘 (수학적 근거)
결과값 집합 $S$에서 90% 신뢰 범위를 다음과 같이 산출합니다.
$$Range_{90\%} = [Percentile(S, 5), Percentile(S, 95)]$$
또한, 변수별 중요도는 입력값과 결과값 사이의 **상관계수(Correlation Coefficient)**를 계산하여 산출합니다.
